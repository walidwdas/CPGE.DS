üöÄ How to Set Up Your Website

  1. Create GitHub Account: Go to github.com and sign up
  2. Create New Repository:
    • Click "New" button
    • Name: "my-resources" (or any name)
    • Select "Public"
    • Check "Add a README file"
    • Click "Create repository"
  3. Create Folders:
    • Click "Add file" ‚Üí "Create new file"
    • Type: math/placeholder.txt
    • Write anything inside and click "Commit"
    • Repeat for: physics/placeholder.txt
  4. Upload This HTML File:
    • Save this page as "index.html"
    • Go back to main repository page
    • Click "Add file" ‚Üí "Upload files"
    • Upload index.html
    • Click "Commit changes"
  5. Upload Your PDF Files:
    • Click on "math" folder
    • Click "Add file" ‚Üí "Upload files"
    • Upload your math PDFs
    • Click "Commit changes"
    • Repeat for "physics" folder
  6. Edit File Names in HTML:

    Find this in the HTML:

    <a href="math/exam1.pdf" class="file-link" target="_blank">

    Change "exam1.pdf" to YOUR actual file name

  7. Enable GitHub Pages:
    • Go to "Settings" tab
    • Click "Pages" in left menu
    • Under "Branch", select "main"
    • Click "Save"
  8. Visit Your Website:

    After 2-3 minutes, go to:
    https://YOUR-USERNAME.github.io/REPOSITORY-NAME/
