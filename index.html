üìù How to Add Your Documents

1 Create GitHub Repository: Go to GitHub.com and create a new repository named "academic-resources" (or any name you prefer). Make sure it's set to PUBLIC.
2 Upload This HTML File: Save this page as "index.html" and upload it to your repository's root folder.
3 Create Folders: In your repository, create two folders: "math" and "physics"
4 Upload PDFs: Upload your PDF files into the respective folders (math PDFs in "math" folder, physics PDFs in "physics" folder)
5 Enable GitHub Pages:
  • Go to your repository Settings
  • Scroll to "Pages" section
  • Under "Source", select "main" branch
  • Click Save
6 Update Links: Edit the index.html file and replace the sample items with your actual files. For each PDF, add: 
<li class="resource-item">
    <a href="math/your-file-name.pdf" download style="text-decoration: none; color: inherit; display: flex; justify-content: space-between; width: 100%;">
        <span class="resource-name">Your File Title</span>
        <span class="resource-type">PDF</span>
    </a>
</li>
7 Access Your Site: Your website will be available at: https://YOUR-USERNAME.github.io/REPOSITORY-NAME/
